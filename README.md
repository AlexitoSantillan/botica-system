# Sistema de Botica - Nova Salud

Sistema web fullstack para la gestión de inventario y ventas de una botica.

## Tecnologías utilizadas

### Frontend
- React
- Axios
- CSS

### Backend
- Node.js
- Express

### Base de datos
- MongoDB

---

## Funcionalidades

- Login de administrador
- Gestión de productos
- Registro de ventas
- Historial de ventas
- Dashboard estadístico
- Alertas de stock bajo
- Alertas de productos próximos a vencer
- Buscador de productos
- CRUD completo

---

## Instalación

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
